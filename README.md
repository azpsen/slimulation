# slimulation
## a gpu-based slime simulation written in c++

## setup
### requirements
- SDL2
- OpenGL3
- GLAD
- Dear ImGUI

### pre-build
the following files should be moved from imgui/backends to imgui/:
- imgui_impl_opengl3.cpp
- imgui_impl_opengl3.h
- imgui_impl_opengl3_loader.h
- imgui_impl_sdl.cpp
- imgui_impl_sdl.h
